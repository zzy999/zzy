1.工具:VC++6.0
2.语言:C语言
3.程序:到目前为止，主要用了6个函数:
  (1).void tip()；主要是给用户一些tip，关于一些公式和一些变量的介绍，
                  让用户更加的熟悉了解这一软件。
  (2).void menu()；主要是功能的选择，制作此界面简洁明了，让用户对软件的功能一目了然。
  (3).void benjin()；此函数是用来求本金的，主要功能是:已知多年后的期望值，
                     反过来求现今所需要多少本金才能达到自己所想的期望值。
  (4).void fuli()；此函数用来求复利终值，主要功能是:知道本金，求多年后的本金和利息的总和，
                   但这里涉及了年复利次数，年复利次数是一年中滚利的次数。
  (5).void danli()；此函数用来求单利计算，这与复利计算的最大区别是:
                    单利计算没有年复利次数，纯粹的计算多年后的本金和。
  (6).main()；这是主函数，运行主要的机制。
4.窗口：由于采用的是VC++6.0，所以没有做成JAVA那样的界面。
        当执行程序时，首先出现在我们面前的是void tip()和void menu()这两个函数的功能，
        起到提示用户的一些操作和让用户了解软件的功能。
        void tip()和void menu()这两个功能只有"结束"程序的时候才会退出我们的视线，
        不然它会在执行每个功能后出现在我们的眼前，这样做的目的是给用户最好的体验，
        省去一些麻烦的操作，方便简洁。
        主要是用了while(1)这个循环和switch()语句来实现这个功能。
